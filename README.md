## Translator API Test

Microsoftの翻訳APIを使って，
日本の昔話を日本語→英語→イタリア語→日本語...と連続して翻訳するテスト。



## 例

```
【原文】

むかしむかし、あるところに、おじいさんとおばあさんが住んでいました。
おじいさんは山へしばかりに、おばあさんは川へせんたくに行きました。
おばあさんが川で洗濯をしていると、ドンブラコ、ドンブラコと、大きな桃が流れてきました。
「おや、これは良いおみやげになるわ」
おばあさんは大きな桃をひろいあげて、家に持ち帰りました。
そして、おじいさんとおばあさんが桃を食べようと桃を切ってみると、なんと、中から元気の良い男の赤ちゃんが飛び出してきました。
「これはきっと、神さまがくださったにちがいない」
子どものいなかったおじいさんとおばあさんは、大喜びです。
桃から生まれた男の子を、おじいさんとおばあさんは桃太郎と名付けました。
桃太郎はスクスク育って、やがて強い男の子になりました。



【翻訳】

一度、おじいちゃんとおばあちゃんが住んでいます。
おじいちゃんおばあちゃんにはよく行きます。
おばあちゃん、ドン ドン blaco、偉大な釣りの川清掃行く blaco。
「ああ、これが起こって良いメモリを持っています。
私の祖母は偉大な釣りをとり、家族を取った。
おじいちゃんとおばあちゃん、回復、桃をカットするから、人間の子供を食べる桃が出てきた。
実際には、実際には神
おじいちゃんとおばあちゃん、子供たちは興奮していた。釣り少年私の祖母と私の両親を名前します。桃太郎は急速に成長し、すぐに強い男になった。



【翻訳(ニューラルネットワーク)】

昔々、彼は老人と彼の祖母を住んでいた。
山のための古い木、おばあちゃんは、洗濯のために川に行きました。
川におばあちゃんがダウンしていた時、ドンブラコ、ドンブラコ、大きな桃が流れていました。
ああ、それはすてきな贈り物だ。
おばあちゃんは大きな桃を取り、彼女の家を取る。
そして、祖父と祖母が桃を食べて桃を切って、元気な子が部屋から出てきました。
これは神によって与えられたにちがいない。
子供のいない祖父と祖母は満足しています。もも、おじいちゃんとおばあちゃんは桃太郎といいます。桃太郎は猫を育て、丈夫な男の子になりました。
```
